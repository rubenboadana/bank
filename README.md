# 🐳 Required tools
1. Docker
2. Maven 3
3. JDK 17

# 👩‍💻🧾 How To Start
1. Build the application: `mvn clean install`
2. Start up Docker compose: `docker-compose up`
3. Go to browser and check the REST api documentation: http://localhost:8000/swagger-ui/index.html
